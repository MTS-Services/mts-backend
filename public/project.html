Project API Documentation

Welcome to the API documentation for managing projects. Below are the details for each endpoint.

1. Create a Project

POST http://192.168.10.40:3000/api/project/create

Request

Body (JSON):

{
    "clientName": "Example Client",
    "opsStatus": "In Progress",
    "salesComments": "Sales comments here",
    "opsleaderComments": "Ops leader comments here",

    "orderedBy": 1,

    "orderAmount": 1000,
    "afterFiverrAmount": 800,
    "bonus": 200,

    "rating": 4,
    "department": "Sales"

2. Get All Projects

POST http://192.168.10.40:3000/api/project

Request

Body (JSON):

{
    "page": 1,
    "limit": 10
}

Response

{
    "totalProjects": 100,
    "projects": [
        {
            "orderId": "2025-04-09T13:20:15.234Z",
            "projectName": "Example Client-2025-04-09T13:20:15.234Z",
            "opsStatus": "In Progress",
            "salesComments": "Sales comments here",
            "opsleaderComments": "Ops leader comments here",
            "sheetLink": "http://example.com/sheet",
            "orderedBy": 1,
            "status": "Active",
            "orderAmount": 1000,
            "afterFiverrAmount": 800,
            "bonus": 200,
            "afterFiverrBonus": 150,
            "rating": 4,
            "department": {
                "name": "Sales"
            }
        }
    ],
    "pagination": {
        "page": 1,
        "limit": 10,
        "total": 100,
        "totalPages": 10
    }
}

Error Response

{
    "error": "An error occurred while fetching projects."
}

3. Update a Project by ID

PUT http://192.168.10.40:3000/api/project/:id

Request

Body (JSON):

{
    "opsStatus": "Completed",
    "salesComments": "Updated sales comments",
    "opsleaderComments": "Updated ops leader comments"
